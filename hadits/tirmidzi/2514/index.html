Kitab: Sifat jahannam

Bab: Api kalian sekarang hanyalah sepertujuh puluh api jahannam

حَدَّثَنَا سُوَيْدٌ أَخْبَرَنَا عَبْدُ اللَّهِ أَخْبَرَنَا مَعْمَرٌ عَنْ هَمَّامِ بْنِ مُنَبِّهٍ عَنْ أَبِي هُرَيْرَةَ
عَنْ النَّبِيِّ صَلَّى اللَّهُ عَلَيْهِ وَسَلَّمَ قَالَ نَارُكُمْ هَذِهِ الَّتِي تُوقِدُونَ جُزْءٌ وَاحِدٌ مِنْ سَبْعِينَ جُزْءًا مِنْ حَرِّ جَهَنَّمَ قَالُوا وَاللَّهِ إِنْ كَانَتْ لَكَافِيَةً يَا رَسُولَ اللَّهِ قَالَ فَإِنَّهَا فُضِّلَتْ بِتِسْعَةٍ وَسِتِّينَ جُزْءًا كُلُّهُنَّ مِثْلُ حَرِّهَا
قَالَ أَبُو عِيسَى هَذَا حَدِيثٌ حَسَنٌ صَحِيحٌ وَهَمَّامُ بْنُ مُنَبِّهٍ هُوَ أَخُو وَهْبِ بْنِ مُنَبِّهٍ وَقَدْ رَوَى عَنْهُ وَهْبٌ

Terjemahan/Arti Hadits:

Telah menceritakan kepada kami Suwaid[1] telah mengabarkan kepada kami Abdullah[2] telah mengabarkan kepada kami Ma'mar[3] dari Hammam bin Munabbih[4] dari Abu Hurairah[5] dari Nabi shallallahu 'alaihi wasallam, beliau bersabda: "Api kalian ini yang kalian gunakan untuk membakar adalah salah satu bagian dari tujuh puluh bagian panasnya api neraka." Mereka bertanya; "Demi Allah, apabila api ini sungguh sudah cukup (panas untuk mengadzab orang yang bermaksiat, lalu mengapa harus ditambah panasnya) wahai Rasulullah?" Beliau menjawab: "Panasnya api neraka melebihi (panasnya api dunia) sembilan puluh sembilan bagian. Semua bagian api (yang berjumlah sembilan puluh sembilan) panasnya adalah seperti panas api neraka." (maksud jawaban Rasulullah adalah adzab Allah harus lebih besar daripada adzab manusia). Abu Isa berkata; 'Hadits ini hadits hasan shahih. Dan Hammam bin Munabbih adalah saudara Wahb bin Munabbih. Dan Wahb telah meriwayatkan hadits darinya."

Sanad Hadits | Catatan Kaki:

  1. [1] Suwaid bin Nashr bin Suwaid, Al Marwaziy, Abu Al Fadlol, Asy Syah, Tabi'ul Atba' kalangan tua, wafat tahun 240 H, hidup di Himsh.
  2. [2] Abdullah bin Al Mubarak bin Wadlih, Al Hanzhaliy Al Marwaziy, Abu 'Abdur Rahman, Abdan, Tabi'ut Tabi'in kalangan pertengahan, wafat tahun 181 H, hidup di Himash, wafat di Herrat.
  3. [3] Ma'mar bin Raosyid, Al Azdiy Al Bashriy, Abu 'Urwah, Tabi'ut Tabi'in kalangan tua, wafat tahun 154 H, hidup di Yaman.
  4. [4] Hammam bin Munabbih bin Kamil bin Syaikh, Ash Shan'aniy Al Abnawiy, Abu 'Uqbah, Tabi'in kalangan tua, wafat tahun 132 H, hidup di Yaman.
  5. [5] Abdur Rahman bin Shakhr, Ad Dawsiy Al Yamaniy, Abu Hurairah, Shahabat, wafat tahun 57 H, hidup di Madinah, wafat di Madinah.
